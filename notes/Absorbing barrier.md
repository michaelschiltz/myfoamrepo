# Absorbing barrier

A ruin state from which a multiplicative trajectory cannot recover — the governing fact behind time-average reasoning. Because the barrier is absorbing, a single crossing ends the trajectory, so the relevant object is not the ensemble average over survivors but the [[Time-average|time-average]] growth rate along the one path that must avoid it.
